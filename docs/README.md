# Docs

GitHub Pages site. Drop HTML, markdown, or Jekyll content here for easy hosting via GitHub's built-in Pages feature.

**Everything in this folder is public.** Including this README. If the repo is private, enabling GitHub Pages on the `docs/` folder still serves its contents to the open web. Do not put anything here you wouldn't want indexed by search engines.

## Use Cases

- **Share experiment results.** Drop an HTML report or interactive demo from `experiments/` here so colleagues or clients can view it in a browser without cloning the repo.
- **Client-facing demos.** Quick way to host a polished version of a POC. Send someone a link instead of a zip file.
- **Project landing page.** A simple index.html that explains what intercom is, with links to the repo.
- **API docs or generated references.** If you auto-generate docs from code, output them here for free hosting.

## What Goes Where

| Content | Location |
|---------|----------|
| Specs and domain knowledge | `specs/` |
| Setup and install instructions | `quick-start.md` (repo root) |
| Anything you want publicly hosted | `docs/` (this folder) |
